# 神经干细胞相关疾病模型

> 聚焦神经退行性疾病、脑损伤、神经发育疾病中的 NSC 异常
> 涵盖体内模型和体外模型

---

## 一、神经退行性疾病

### 1.1 阿尔茨海默病 (AD)

| 方面 | 表现 |
|------|------|
| **NSC 变化** | 海马 SGZ 神经发生显著下降（早期即出现） |
| **分子机制** | Aβ 寡聚体抑制 Wnt 信号；Tau 病理破坏 NSC 分裂 |
| **动物模型** | 5xFAD, APP/PS1, 3xTg-AD |
| **体外模型** | hiPSC→NSC→皮层神经元，Aβ 处理 |
| **潜在靶点** | Wnt 激活、GSK3β 抑制、BDNF 补充 |

### 1.2 帕金森病 (PD)

| 方面 | 表现 |
|------|------|
| **NSC 变化** | SVZ 神经发生下降（黑质多巴胺能神经元减少的补偿代偿） |
| **分子机制** | α-synuclein 聚集损伤 NSC 增殖；氧化应激 |
| **动物模型** | MPTP, 6-OHDA, α-synuclein AAV |
| **体外模型** | 人 iPSC→NSC→中脑多巴胺能神经元 |
| **潜在靶点** | SHH 激活（多巴胺神经元诱导）、抗氧化 |

### 1.3 亨廷顿病 (HD)

| 方面 | 表现 |
|------|------|
| **NSC 变化** | SVZ 神经发生初始增强，后衰竭 |
| **分子机制** | mHtt 破坏 Wnt/β-catenin 和 Notch 信号 |
| **动物模型** | R6/2, BACHD, Q175 |
| **潜在靶点** | 增强 SHH 通路，神经营养因子支持 |

## 二、脑损伤

### 2.1 缺血性脑卒中

| 方面 | 表现 |
|------|------|
| **NSC 变化** | SVZ 和 SGZ 反应性增殖（48h 内开始，7-14d 达峰） |
| **分子机制** | 炎症因子 (IL-6, TNF-α) → Notch/Hippo 信号改变 |
| **实验模型** | MCAO（大脑中动脉闭塞）、PT（光血栓） |
| **治疗策略** | FGF2、EGF 激活内源 NSC；移植 eNSC |

### 2.2 创伤性脑损伤 (TBI)

| 方面 | 表现 |
|------|------|
| **NSC 变化** | 初始抑制 → 反应性增殖 → 长期衰竭 |
| **信号通路** | ROCK 通路失调（见 Yao 2026 PMID: 40749978） |
| **实验模型** | CCI（控制性皮质冲击）、FPI（流体冲击） |
| **干预** | ROCK 抑制 + 环境富集 |

### 2.3 脊髓损伤

| 方面 | 表现 |
|------|------|
| **脊髓 NSC** | 室管膜细胞→反应性 NSC→星形胶质细胞为主 |
| **限制因素** | 髓鞘抑制蛋白 (Nogo, MAG, OMgp) 阻碍神经元分化 |
| **潜在靶点** | Notch 抑制 + BMP 拮抗促进神经元再生 |

## 三、神经发育疾病

### 3.1 自闭症谱系障碍 (ASD)

| 方面 | 表现 |
|------|------|
| **NSC 异常** | 早期过度增殖（大头脑），后期突触修剪缺陷 |
| **通路** | mTOR 通路过度激活；Wnt 信号异常 |
| **体外模型** | ASD 患者 iPSC→NSC→类脑器官 |
| **基因** | CHD8, PTEN, SHANK3, TSC1/2 |

### 3.2 小头畸形

| 方面 | 表现 |
|------|------|
| **NSC 异常** | 早期神经前体细胞增殖不足，过早分化 |
| **通路** | 中心体/有丝分裂缺陷；DNA 损伤反应异常 |
| **基因** | ASPM, WDR62, CDK5RAP2, CENPJ |
| **体外模型** | 患者 iPSC→大脑类器官 |

### 3.3 智力障碍

| 特征 | 相关基因 | NSC 表型 |
|------|---------|---------|
| 突触可塑性缺陷 | FMR1 (FXS), MECP2 (RTT) | 突触形成和成熟异常 |
| 神经元迁移异常 | DCX, RELN, LIS1 | 放射状迁移障碍 |
| 中间神经元缺陷 | ARX, DLX 家族 | GABA 能神经元减少 |

## 四、体外模型系统

| 模型 | 培养周期 | 优势 | 局限 |
|------|---------|------|------|
| **神经球 (Neurosphere)** | 5-10d | 经典、简单、高通量 | 混合细胞群体，核心缺氧 |
| **单层 NSC (Monolayer)** | 持续扩增 | 均质、可成像 | 丧失 3D 生态位 |
| **SVZ/SGZ 切片** | 1-7d | 保留生态位结构 | 存活时间短 |
| **大脑类器官** | 1-6 月 | 发育模拟、复杂结构 | 变异性大、无血管 |
| **区域特异性类器官** | 1-6 月 | 特定脑区模拟 | 技术难度高 |
| **微流控芯片 (Brain-on-chip)** | 7-28d | 可控微环境 + 流体 | 通量低 |
| **共培养体系** | 7-14d | 研究细胞间互作 | 人为搭建 |

### 推荐：虚拟细胞智能体适用的体外模型数据库

| 数据库 | 数据内容 | 适用场景 |
|--------|---------|---------|
| GEO (NCBI) | 神经干细胞转录组 | 差异表达分析、调控网络重建 |

| Single Cell Portal (Broad) | 大脑单细胞图谱 | 细胞类型注释、稀有细胞鉴定 |

## 五、NSC 移植治疗现状

| 疾病 | 临床试验阶段 | 细胞来源 | 结果 |
|------|------------|---------|------|
| 帕金森病 | Phase I/II | 胎儿腹侧中脑、iPSC-多巴胺神经元 | 部分改善、异质性高 |
| 脊髓损伤 | Phase I/II | 胚胎 NSC、iPSC-NSC | 安全性好，疗效有限 |
| 缺血性卒中 | Phase I/II | CTX0E03 永生化 NSC | 部分功能改善 |
| 肌萎缩侧索硬化 (ALS) | Phase I/II | 脊髓 NSC | 安全性验证 |
| 阿尔茨海默病 | 临床前 | NSC 移植 + 神经营养因子释放 | 动物模型获益 |

**AI 在移植中的角色**：预测最佳移植细胞类型、筛选最有利的微环境条件、预测免疫排斥风险。
