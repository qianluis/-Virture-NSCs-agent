# 神经干细胞（NSC）标记物基因库

> 来源整理自 CellMarker 2.0, PubMed 综述, 单细胞转录组研究
> 最后更新: 2026.06

## 一、核心标记物分类

### 1.1 经典 NSC 标记物（跨物种保守）

| 基因 | 蛋白 | 定位 | 物种保守性 | 说明 |
|------|------|------|-----------|------|
| **NES** | Nestin | 中间丝蛋白 | 高 | NSC 最经典标记物，发育期和成年 NSC 均表达 |
| **SOX2** | SRY-box 2 | 转录因子 | 极高 | 干性维持核心 TF，随分化下调 |
| **PAX6** | Paired box 6 | 转录因子 | 高 | 大脑皮层 NSC 标记，径向胶质细胞标记 |
| **GFAP** | Glial fibrillary acidic protein | 中间丝蛋白 | 高 | 成年 NSC（SVZ 和 SGZ B 细胞）标记 |
| **VIM** | Vimentin | 中间丝蛋白 | 高 | 未成熟 NSC 表达，随分化下降 |
| **PROM1** | CD133 / Prominin-1 | 跨膜糖蛋白 | 高 | 人 NSC 富集标记，与 Notch 通路相关 |
| **FABP7** | BLBP (Brain lipid-binding protein) | 胞质蛋白 | 高 | 径向胶质细胞/NSC 标记 |
| **GLI3** | GLI family zinc finger 3 | 转录因子 | 高 | SHH 通路下游，NSC 维持 |

### 1.2 成年 NSC 亚型标记物

| 亚型 | 标记物组合 | 定位 | 特征 |
|------|-----------|------|------|
| **SVZ B 细胞 (qNSC)** | GFAP⁺ / NES⁺ / SOX2⁺ / EGFR⁻ | 侧脑室室下带 | 静息态 NSC，具长突起接触室管膜 |
| **SVZ B 细胞 (aNSC)** | GFAP⁺ / NES⁺ / SOX2⁺ / EGFR⁺ | 侧脑室室下带 | 激活态 NSC，进入细胞周期 |
| **SVZ C 细胞** | ASCL1⁺ / EOMES⁺ / Ki-67⁺ | SVZ | 快速扩增前体细胞 |
| **SVZ A 细胞** | DCX⁺ / TUBB3⁺ / PSA-NCAM⁺ | RMS | 成神经细胞，沿吻侧迁移流迁移 |
| **SGZ 1 型 NSC** | GFAP⁺ / NES⁺ / SOX2⁺ | 海马齿状回 | 形态类似径向胶质 |
| **SGZ 2 型 NSC** | SOX2⁺ / ASCL1⁺ / GFAP⁻ | 海马齿状回 | 过渡扩增前体细胞 |
| **SGZ 3 型 NSC** | DCX⁺ / TUBB3⁺ / CALB2⁺ / NEUN⁻ | 海马齿状回 | 未成熟神经元 |

### 1.3 发育期 NSC 标记物

| 阶段 | 标记物 | 说明 |
|------|--------|------|
| **神经板期** | SOX1, SOX2, SOX3, NES | 最早神经外胚层标记 |
| **神经管期** | PAX6, EMX2, OTX1/2 | 前后轴模式化 |
| **径向胶质** | GFAP, FABP7, GLAST (SLC1A3) | 兼具 NSC 和支架功能 |
| **中间前体细胞** | TBR2 (EOMES), NEUROG2 | 二次扩增阶段 |
| **未成熟神经元** | DCX, TUBB3, STMN2 | 迁移和早期分化 |

## 二、分化谱系标记物

### 2.1 神经元分化

| 阶段 | 标记物 | 功能 |
|------|--------|------|
| 神经决定 | **NEUROG1/2, ASCL1, ATOH1** | 促神经发生的 bHLH 转录因子 |
| 神经元规范 | **NEUROD1/2/4/6** | 神经元分化执行 TF |
| 轴突生长 | **STMN2 (SCG10), GAP43** | 生长锥动态 |
| 突触成熟 | **SYN1, SNAP25, DLG4 (PSD95)** | 突触小泡 / 支架蛋白 |
| 成熟神经元 | **RBFOX3 (NeuN), MAP2** | 成熟神经元标志 |

### 2.2 星形胶质细胞分化

| 标记物 | 出现阶段 | 功能 |
|--------|---------|------|
| **GFAP** | 成熟星形胶质细胞 | 中间丝，结构支撑 |
| **S100B** | 成熟星形胶质细胞 | Ca²⁺ 结合蛋白 |
| **ALDH1L1** | 各类星形胶质细胞 | 叶酸代谢 |
| **AQP4** | 成熟星形胶质细胞 | 水通道，血管周足 |
| **SLC1A3 (GLAST)** | 径向胶质→星形胶质细胞 | 谷氨酸转运体 |

### 2.3 少突胶质细胞分化

| 阶段 | 标记物 | 说明 |
|------|--------|------|
| OPC | **PDGFRA, CSPG4 (NG2), SOX10** | 少突前体细胞 |
| 前少突胶质 | **OLIG1, OLIG2, NKX2-2** | 谱系决定 TF |
| 未成熟少突胶质 | **MBP, PLP1, CNP** | 髓鞘蛋白开始表达 |
| 成熟少突胶质 | **MOG, MAG, CLDN11** | 功能性髓鞘形成 |

## 三、物种特异性差异（重要！）

| 特征 | 小鼠/大鼠 | 人 |
|------|----------|---|
| 成年 SVZ 神经发生 | 活跃终生 | 出生后骤降，成人极低 |
| 成年 SGZ 神经发生 | 活跃终生 | 存在但争议大，年龄相关下降显著 |
| 体外扩增 | 神经球悬浮培养易扩增 | 贴壁单层培养更稳定 |
| 标记物 POU5F1 | 几乎不表达 | 部分研究中可检测到 |
| 细胞周期 | ~12-18h | ~24-36h |
| OCT4 依赖性 | 低 | 人 iPS→NSC 诱导中重要 |
