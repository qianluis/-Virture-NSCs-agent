# 神经干细胞关键信号通路

> 覆盖发育期和成年 NSC 的 6 大核心通路
> 通路拓扑基于 KEGG + Reactome + 文献综述

---

## 一、Notch 通路 — NSC 维持的核心开关

### 通路概要

```
配体细胞                      NSC
┌──────────┐           ┌──────────────────┐
│  DLL1/4  │───Delta──→│   Notch1/2/3/4   │
│  JAG1/2  │───Jagged─→│   (跨膜受体)      │
└──────────┘           └────────┬─────────┘
                                │ 配体结合触发
                                ▼
                         γ-secretase 切割
                                │
                                ▼
                    NICD (Notch 胞内域)
                                │
                        入核 + RBPJ
                                │
                                ▼
              ┌──────────────────────────┐
              │  Hes1 / Hes5 / Hey1/2    │ ← 转录激活
              └──────────────────────────┘
                        │
                        ▼
              ┌──────────────────────────┐
              │  ASCL1 ↓   NEUROG2 ↓     │ ← 神经发生抑制
              │  SOX2 →    NSC 维持       │
              └──────────────────────────┘
```

### 关键节点

| 节点 | 类型 | 功能 |
|------|------|------|
| DLL1/JAG1 | 配体 | 相邻细胞激活 Notch |
| Notch1 | 受体 | NSC 主要 Notch 亚型 |
| γ-secretase | 蛋白酶复合体 | NICD 释放的限速酶 |
| RBPJ (CSL) | 转录共抑制因子 | NICD 入核后转为激活因子 |
| HES1/HES5 | bHLH 转录抑制因子 | 抑制 pro-neural 基因 |
| ASCL1 (Mash1) | 靶基因 | Notch→HES 抑制 ASCL1，阻止分化 |

### 在 NSC 中的功能
- **维持 NSC 池**: Notch 激活 → HES 表达 → 抑制神经发生 → 维持 NSC 干细胞状态
- **侧向抑制**: Notch 介导的相邻细胞命运抉择（一个细胞分化为神经元，周边细胞维持 NSC）
- **发育期**: 放射状胶质细胞 → 神经元的时间顺序生成
- **成年 NSC**: SVZ B 细胞中 Notch 活性维持静息态

### 干预效应预测

| 干预 | 预期效果 | 文献支持强度 |
|------|---------|------------|
| Notch1 KO | NSC 过早分化，NSC 池耗竭 | ⭐⭐⭐⭐⭐ |
| γ-secretase 抑制 (DAPT) | NSC 向神经元方向分化增加 | ⭐⭐⭐⭐⭐ |
| DLL1 过表达 | 相邻细胞 Notch 激活增强，NSC 维持 | ⭐⭐⭐⭐ |
| HES1 过表达 | 神经发生全面抑制 | ⭐⭐⭐⭐ |
| ASCL1 过表达 | 强迫 NSC 进入神经发生程序 | ⭐⭐⭐⭐⭐ |

## 二、Wnt/β-catenin 通路 — 神经发生驱动器

### 通路概要

```
Wnt 配体 (Wnt3a, Wnt7a, Wnt1)
        │
        ▼
┌──────────────────┐
│  Frizzled + LRP5/6 │ ← 膜受体复合体
└────────┬─────────┘
         │
         ▼
┌──────────────────┐
│  DVL 激活        │
│  (Dishevelled)   │
└────────┬─────────┘
         │
         ▼ 破坏复合体失活
┌─────────────────────────────────────┐
│  [AXIN / APC / GSK3β / CK1] → 解体  │
│  β-catenin 不再被磷酸化降解          │
└─────────────────────────────────────┘
         │
         ▼
β-catenin 积累 → 入核
         │
         ▼
┌──────────────────┐
│  TCF/LEF          │ ← β-catenin 结合后解除抑制
│  + 靶基因激活     │
└──────────────────┘
         │
         ▼
  NEUROG2 / ASCL1 / N-myc / Cyclin D1
  → 神经发生 + 细胞周期激活
```

### 关键节点

| 节点 | 类型 | 功能 |
|------|------|------|
| Wnt3a/Wnt7a | 配体 | 经典 Wnt 信号 |
| LRP6 | 共受体 | 配体结合必须 |
| GSK3β | 激酶 | β-catenin 磷酸化 → 降解 |
| β-catenin (CTNNB1) | 转录共激活因子 | 核心效应分子 |
| TCF4/LEF1 | 转录因子 | 结合 β-catenin 调控靶基因 |
| AXIN2 | 支架蛋白，负反馈 | Wnt 靶基因之一 |

### 在 NSC 中的功能
- **发育期**: 前后轴模式化，端脑背腹侧分化
- **成年 NSC**: SGZ 中 Wnt3a 维持神经发生；SVZ 中 Wnt 信号调节 NSC 激活
- **增殖 vs 分化**: 胞质 β-catenin → 增殖；核 β-catenin + TCF → 分化
- **与 Notch 交叉**: β-catenin 直接调控 HES1 表达

### 干预效应预测

| 干预 | 预期效果 | 文献支持强度 |
|------|---------|------------|
| β-catenin 组成型激活 | NSC 增殖增加，但分化异常 | ⭐⭐⭐⭐ |
| GSK3β 抑制 (CHIR99021) | β-catenin 积累，NSC 扩增 | ⭐⭐⭐⭐⭐ |
| Wnt3a 处理 | SGZ 神经发生增加 | ⭐⭐⭐⭐ |
| Wnt 拮抗剂 (DKK1) | 神经发生抑制 | ⭐⭐⭐⭐ |
| TCF4 KO | 海马神经发生缺陷 | ⭐⭐⭐ |

## 三、SHH 通路 — 腹侧模式化与 NSC 维持

### 通路概要

```
SHH (Sonic Hedgehog)
        │
        ▼
┌──────────────────┐
│  PTCH1 (受体)     │ ← 无 SHH 时抑制 SMO
│  SMO (GPCR 样)    │
└────────┬─────────┘
  SHH 结合 PTCH1 → 解除对 SMO 的抑制
         │
         ▼
┌──────────────────┐
│  GLI 转录因子处理  │
│  (GLI1/2/3)      │
└────────┬─────────┘
         │
         ▼
  GLI-A (激活型)     GLI-R (抑制型)
         │                │
         ▼                ▼
   靶基因激活        靶基因抑制
  (GLI1, PTCH1,    (Hhip 等)
   MYCN, CCND2)
```

### 关键节点

| 节点 | 类型 | 功能 |
|------|------|------|
| SHH | 配体 | 腹侧神经管模式化 |
| PTCH1 | 12 次跨膜受体 | SMO 的负调控因子 |
| SMO | 7 次跨膜 GPCR 样 | 信号传递必须 |
| SUFU | 胞质抑制因子 | GLI 的核质穿梭调节 |
| GLI1 | 转录激活因子 | SHH 信号指示器 |
| GLI2 | 双重功能 | 兼具激活和抑制形式 |
| GLI3 | 主要抑制因子 | 无 SHH 时加工为截短抑制形式 |

### 在 NSC 中的功能
- **发育期**: 神经管腹侧模式化（底板→运动神经元→中间神经元梯度）
- **成年 NSC**: SVZ 中 SHH 维持 NSC 自我更新；损伤后 SHH 激活 NSC
- **小脑**: SHH 是颗粒神经元前体的主要有丝分裂原
- **与 Wnt 交叉**: SHH 和 Wnt 共同控制背腹轴

### 干预效应预测

| 干预 | 预期效果 | 文献支持强度 |
|------|---------|------------|
| SHH KO | 腹侧神经管缺失，NSC 减少 | ⭐⭐⭐⭐⭐ |
| SMO 激活 (SAG) | NSC 增殖增加 | ⭐⭐⭐⭐ |
| SMO 抑制 (Cyclopamine) | 腹侧 NSC 分化异常 | ⭐⭐⭐⭐⭐ |
| GLI3 KO | 背侧 NSC 扩增 | ⭐⭐⭐ |
| SUFU KO | 组成型 SHH 信号，NSC 过度增殖 | ⭐⭐⭐ |

## 四、BMP 通路 — NSC 静息与胶质分化

### 通路概要

```
BMP2/4/7, GDF
        │
        ▼
┌──────────────────┐
│  BMPR1a/1b + BMPR2│ ← 丝/苏氨酸激酶受体
└────────┬─────────┘
         │
         ▼
┌──────────────────┐
│  R-SMAD (SMAD1/5/8) │
│  + Co-SMAD (SMAD4)  │
└────────┬─────────┘
  R-SMAD 磷酸化 → 与 SMAD4 形成复合体
         │
         ▼
        入核 → 靶基因
         │
         ▼
  ID1/2/3/4, HES5, GFAP
  → 抑制神经发生，促进星形胶质细胞分化
```

### 关键节点

| 节点 | 类型 | 功能 |
|------|------|------|
| BMP2/4 | 配体 | 神经诱导和背侧化 |
| BMPR1a (ALK3) | I 型受体 | NSC 中主要 BMP 受体 |
| SMAD1/5/8 | R-SMAD | 激活形式的效应分子 |
| SMAD4 | Co-SMAD | 所有 TGF-β 信号共用 |
| SMAD7 | I-SMAD | 负反馈抑制因子 |
| ID1-4 | HLH 抑制因子 | 结合 E 蛋白，阻断 pro-neural bHLH |

### 在 NSC 中的功能
- **发育期**: 神经管背侧 BMP 梯度 → 神经嵴和背部中间神经元
- **成年 SVZ**: BMP 信号维持 NSC 静息态，BMP 拮抗剂（Noggin/Chordin）促进激活
- **胶质分化**: BMP 是星形胶质细胞分化的强诱导剂
- **与 Notch 交叉**: BMP→SMAD→ID 与 Notch→HES 通路协同

### 干预效应预测

| 干预 | 预期效果 | 文献支持强度 |
|------|---------|------------|
| BMP4 处理 | NSC 增殖抑制，星形胶质分化增加 | ⭐⭐⭐⭐⭐ |
| Noggin (BMP 拮抗剂) | NSC 激活增加，神经发生增强 | ⭐⭐⭐⭐ |
| SMAD4 KO | 神经发生增加，胶质生成减少 | ⭐⭐⭐⭐ |
| ID1 过表达 | 促进 NSC 维持 | ⭐⭐⭐ |

## 五、MAPK/ERK 通路 — 增殖与生存

### 通路概要

```
生长因子 (EGF, FGF2, PDGF)
        │
        ▼
┌──────────────────┐
│  RTK (EGFR, FGFR) │ ← 受体酪氨酸激酶
└────────┬─────────┘
         │
         ▼
┌──────────────────┐
│  RAS → RAF       │
└────────┬─────────┘
         │
         ▼
┌──────────────────┐
│  MEK1/2          │
└────────┬─────────┘
         │
         ▼
┌──────────────────┐
│  ERK1/2          │ ← 磷酸化多种底物
└────────┬─────────┘
         │
         ▼
  Elk-1, c-Myc, RSK, STAT3
  → 增殖 / 生存 / 迁移
```

### 关键节点

| 节点 | 类型 | 功能 |
|------|------|------|
| EGFR | RTK | 激活态 NSC (aNSC) 标记 |
| FGFR1-3 | RTK | NSC 增殖和神经发生的双重调节 |
| KRAS | 小 G 蛋白 | EGFR→RAS→RAF 级联 |
| BRAF | 激酶 | RAF 在 NSC 中的主要亚型 |
| ERK1/2 | 激酶 | 效应 MAPK，磷酸化众多底物 |

### 在 NSC 中的功能
- **发育期**: FGF2 是前脑 NSC 的有丝分裂原；EGF 在晚期神经发生中起主导作用
- **SVZ**: EGF→EGFR 信号是激活态 aNSC 的标志
- **体外培养**: EGF + FGF2 是 NSC 培养基的基本配方
- **与 STAT3 交叉**: ERK→STAT3 在 NSC 自我更新中协同

### 干预效应预测

| 干预 | 预期效果 | 文献支持强度 |
|------|---------|------------|
| EGF 撤除 | NSC 增殖下降，分化增加 | ⭐⭐⭐⭐⭐ |
| FGF2 处理 | NSC 增殖和神经发生增加 | ⭐⭐⭐⭐⭐ |
| MEK 抑制 (U0126) | NSC 增殖抑制 | ⭐⭐⭐⭐ |
| EGFR 激活突变 | NSC 异常增殖（如 GBM） | ⭐⭐⭐⭐ |
| FGFR 抑制 | 前脑 NSC 减少 | ⭐⭐⭐ |

## 六、Hippo/YAP 通路 — 机械力与细胞命运

### 通路概要

```
细胞密度 / 机械力 / GPCR
        │
        ▼
┌──────────────────┐
│  MST1/2 → LATS1/2 │ ← 激酶级联
└────────┬─────────┘
         │
         ▼
┌──────────────────┐
│  YAP/TAZ          │ ← 转录共激活因子
│  (磷酸化→滞留胞质) │
└────────┬─────────┘
  去磷酸化 → 入核
         │
         ▼
┌──────────────────┐
│  TEAD1-4          │
│  + 靶基因激活     │
└──────────────────┘
         │
         ▼
  CYR61, CTGF, ANKRD1, BIRC5
  → 增殖 / 抗凋亡 / SOX2 维持
```

### 关键节点

| 节点 | 类型 | 功能 |
|------|------|------|
| MST1/2 | Ste20 样激酶 | 核心激酶，磷酸化 LATS |
| LATS1/2 | AGC 激酶 | YAP/TAZ 磷酸化 |
| YAP1 | 转录共激活因子 | Hippo 主要效应器 |
| TAZ (WWTR1) | 转录共激活因子 | YAP 旁系同源物 |
| TEAD1-4 | 转录因子 | YAP/TAZ 的 DNA 结合搭档 |
| AMOT | 脚手架蛋白 | YAP 在紧密连接处滞留 |

### 在 NSC 中的功能
- **皮质发育**: YAP 维持放射状胶质细胞的顶端过程
- **成年 NSC**: 高密度/接触抑制 → Hippo 激活 → YAP 失活 → NSC 静息
- **SVZ 生态位**: 室管膜细胞纤毛感知流体剪切力 → YAP 调节 NSC
- **体外**: NSC 密度调控 YAP 核定位水平

### 干预效应预测

| 干预 | 预期效果 | 文献支持强度 |
|------|---------|------------|
| YAP 过表达 | NSC 扩增增加 | ⭐⭐⭐⭐ |
| YAP KO | NSC 过早分化 | ⭐⭐⭐ |
| LATS1/2 抑制 | YAP 激活，NSC 增殖 | ⭐⭐⭐ |
| TEAD 显性负 | NSC 增殖下降 | ⭐⭐⭐ |

## 七、通路交叉调控网络

```
                    ┌─────────────────────────┐
                    │     BMP 通路             │
                    │  (维持静息 / 胶质分化)    │
                    └────────┬──┬──────────────┘
                             │  │
                      SMAD-ID │  │ BMP 拮抗: Noggin
                             │  │
                             ▼  │
┌───────────────┐    ┌─────────────────┐    ┌──────────────────┐
│  Notch 通路    │◄───│    NSC 命运决策    │───►│  SHH 通路         │
│  (维持干性)    │    │                   │    │  (腹侧模式化)     │
│  HES→ASCL1↓   │───►│  [增殖 vs 分化]   │◄───│  GLI→MYCN/CCND2  │
└───────────────┘    └────────┬─────────┘    └──────────────────┘
                             │ │ │
               ┌─────────────┘ │ └─────────────┐
               ▼               ▼               ▼
        ┌────────────┐  ┌────────────┐  ┌────────────┐
        │  MAPK/ERK  │  │  Wnt 通路  │  │  Hippo/YAP │
        │ (增殖/生存) │  │ (神经发生) │  │ (机械力/接触)│
        └────────────┘  └────────────┘  └────────────┘
```

### 已知交叉节点

| 交叉点 | 通路 1 | 通路 2 | 机制 |
|--------|--------|--------|------|
| HES1 | Notch | Wnt | HES1 直接抑制 TCF4 转录 |
| GLI3 | SHH | Wnt | GLI3 与 β-catenin / TCF 物理互作 |
| ID1 | BMP | Notch | ID1 与 HES 竞争 E 蛋白 |
| β-catenin | Wnt | Hippo | β-catenin 与 YAP 竞争 TCF 结合位点 |
| SMAD1 | BMP | Wnt | SMAD1-β-catenin-LEF 三体复合物 |
| ERK | MAPK | Hippo | ERK 磷酸化 LATS 影响 YAP 活性 |
| STAT3 | MAPK | Notch | STAT3 与 NICD 协同激活 HES |
